Para PC (Windows - PowerShell) 
instalar Node.js
Instalar Git
Instalar FFmpeg (desde PowerShell) comando: "winget install --id=Gyan.FFmpeg"

Desde PowerShell clonar repositorio
comandos:
1- git clone https://github.com/nitse007/Bot-WhatsApp.git (copiar y pegar)
2- cd Bot-WhatsApp (ruta del bot)
3- npm install
4- npm start
